## 今天的目标

让「复制粘贴 + 窗口管理」成为肌肉记忆。

⏱️ 预计时间：60 分钟　🎯 难度：★☆☆☆☆

---

## 🔧 今日工具箱

今天只学这 8 个快捷键，看一遍就行，不用背：

**复制粘贴五件套 ⭐**

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + C` | 复制 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + X` | 剪切 |
| `Ctrl + Z` | 撤销（后悔药）|
| `Ctrl + A` | 全选 |

**窗口管理三件套 ⭐**

| 快捷键 | 作用 |
|--------|------|
| `Alt + Tab` | 在打开的窗口之间切换 |
| `Win + ← / →` | 把窗口贴到屏幕左/右半边 |
| `Win + D` | 瞬间显示桌面（再按一次回到原状态）|

## 🎯 任务清单

把鼠标放远一点，从这一刻开始，只用键盘。

### 任务 1.1 · 桌面文字整理（10 分钟）

**场景：** 你桌面上有一些零散的文字记录，需要整合到一个文档里。

1. 在桌面新建一个记事本文件，起名「今日整理.txt」
2. 打开它，输入三行任意文字
3. 用 `Ctrl + A` 全选，用 `Ctrl + C` 复制
4. 回车换三次行
5. 用 `Ctrl + V` 粘贴 3 次
6. 用 `Ctrl + Z` 撤销 2 次，看看效果
7. 用 `Ctrl + S` 保存

✅ **通关标准：全程没有碰鼠标**

---

### 任务 1.2 · 双窗口对照（10 分钟）

**场景：** 你需要一边看资料一边写笔记。

1. 打开浏览器，随便访问一个网页
2. 再打开刚才那个记事本
3. 点一下浏览器窗口，按 `Win + ←`，浏览器贴到左半边
4. 屏幕右半边选记事本，按回车
5. 试试 `Alt + Tab`，在两个窗口间切换

✅ **通关标准：能熟练用 Win+←/→ 分屏，用 Alt+Tab 切换**

---

### 任务 1.3 · 紧急隐藏术（5 分钟）

**场景：** 老板突然走过来。

1. 打开几个无关的窗口
2. 按 `Win + D`，屏幕瞬间变成桌面
3. 再按一次 `Win + D`，所有窗口回来了
4. 重复 5 次，练到反应不到 1 秒

✅ **通关标准：看见「老板来了」四个字，手能在 1 秒内按下 Win+D**

---

### 任务 1.4 · 文件批量重命名（15 分钟）

**场景：** 下载文件夹里有一堆乱七八糟的文件名。

1. 按 `Win + E` 打开文件资源管理器
2. 进入你的下载文件夹
3. 用方向键选中第一个文件
4. 按 `F2`（Yoga 笔记本按 Fn+F2），进入重命名模式
5. 输入新名字，按回车确认
6. 按 `↓` 选下一个，继续按 F2
7. 重命名 5 个文件

✅ **通关标准：全程不用鼠标**

---

## 🆘 卡住怎么办

**Win + ← 没反应？**
可能是 Lenovo Vantage 里禁用了 Win 键。替代方案：用 Alt+Tab 切换。

**不小心用了鼠标？**
没关系，继续练。把鼠标放到键盘够不着的地方，逼自己不去摸。
